## Overview
A full-stack gadget resale platform which allows users to view, post and delete posts for gadget resale.

## Tech Stack
TypeScript, React, Node.js, Express, MongoDB

## Features
- Create gadget posts
- View gadget details
- Pagination
- REST APIs
- UI with glassmorphic design elements and parallax effect

## Run Locally

git clone https://github.com/aparn-gupta/gadgets.git
npm install
npm start
